Melodiy




  • Venues: Easily find bands and book live music in your town
  • Artists: Get more gigs and help build real life community
  • Leverage existing social media and artist platforms

  • Book live music | melodiy.org

Melodiy is still under active development and has not yet been released to the public


Technologies being used Used

  • Angular.js client side Javascript framework
  • Node and Express server side Javascript framework
  • PostgreSQL
  • Bootstrap Responsive CSS
  • Facebook, Foursquare, Last.fm APIs
