Let's Eat


LetsEat.io is a real time collaborative single page web application for deciding on a place to eat amongst a group of people.


Here's how it works

  • Users create a room with the name of their company or group
  • Next the group is given a URL which they can share with their friends or colleagues
  • Users can search Yelp for whatever kind of food or restaurant they are interested in
  • Nearby results are displayed on a map
  • Users can chat in real time and vote on where they want to go
  • An adjustable clock counts down the time until its time to eat, at which point users are emailed the decision

Technologies Used:

  • Meteor Client and Server side Javascript Framework
  • MongoDB NoSQL Database
  • Handelbars Templating
  • Yelp API
  • Leaflet for Mapping
  • Git
