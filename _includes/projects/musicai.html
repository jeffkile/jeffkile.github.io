Music Artificial Intelligence


Music creation software which uses artificial intelligence to compose music.


Features:

  • Multiple cooperative independent agents
  • Music composed in MIDI standards for portability and interesting I/O possibilities
  • The algorithm attempts to emulate how Jazz musicians improvise or how birds flock in that no one agent has complete control yet collectively they determine how the music progresses

History: This project initially began as a semester long project for my Advanced Projects in Artificial Intelligence course at UC Irvine.


Future: Current work is being done on connecting the software to real world I/O such as synthesizers and drum machines. Initially to allow for better sounding sounds then the current software synths, but long term the goal is to allow user input to communicate and play in real time with the AI agents. Additionally, more advanced music theory, genres and visualizations are also planned.


Technology:

  • Multi Threaded Java
  • Java JSound
  • MIDI

The source has been open sourced and contributions are welcome on Github
